# AI Reader

AI Reader is a full-stack AI-powered document reading and analysis application. It includes an Electron desktop app, Node.js backend, Python AI service, and shared TypeScript types.

## Features

- AI-powered document reading
- Document processing and analysis
- Electron desktop application
- Node.js backend API
- Python AI service
- Shared TypeScript types
- Modular project architecture
- Environment-based configuration

## Tech Stack

| Layer | Technology |
|---|---|
| Desktop App | Electron |
| Backend | Node.js |
| AI Service | Python |
| Shared Types | TypeScript |
| Package Manager | npm |
